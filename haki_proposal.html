üìù KARYA 1: SMART EXAM MANAGER

üéì Smart Exam Manager v5.1 FIXED

üìå Deskripsi Singkat

Smart Exam Manager adalah sistem manajemen ujian berbasis web yang memungkinkan guru untuk membuat, mengelola, dan mendistribusikan soal ujian digital dengan mudah. Sistem ini dirancang untuk menciptakan pengalaman ujian yang aman, terstruktur, dan mudah digunakan baik oleh guru maupun siswa.

üéØ Tujuan dan Manfaat

  • Efisiensi Waktu: Mengurangi waktu pembuatan dan distribusi soal ujian hingga 70%
  • Keamanan Ujian: Sistem anti-kecurangan dengan deteksi pelanggaran otomatis
  • Fleksibilitas: Mendukung berbagai jenis soal (pilihan ganda, soal bergambar, multiple correct answers)
  • Portabilitas: File ujian dalam format HTML tunggal, mudah didistribusikan
  • Integrasi: Terintegrasi dengan QR Scanner Dashboard untuk pengumpulan hasil

QR Code Result System: Hasil ujian dikodekan dalam QR Code yang tidak dapat dimanipulasi, memastikan integritas data.

üì± KARYA 2: QR SCANNER DASHBOARD

üìä QR Scanner Dashboard v5.1 FIXED

üìå Deskripsi Singkat

QR Scanner Dashboard adalah aplikasi web yang memungkinkan guru untuk mengumpulkan dan mengolah hasil ujian siswa melalui pemindaian QR Code. Sistem ini menjadi pasangan sempurna untuk Smart Exam Manager, menciptakan ekosistem ujian digital yang lengkap dari pembuatan hingga pengolahan hasil.

üéØ Tujuan dan Manfaat

  • Pengumpulan Cepat: Scan QR Code langsung dari layar smartphone siswa
  • Verifikasi Otomatis: Validasi data hasil ujian secara real-time
  • Analisis Instan: Statistik kelas otomatis (rata-rata, tertinggi, terendah)
  • Export Data: Export ke Excel/CSV untuk pengolahan lebih lanjut
  • Multi-Format Support: Kompatibel dengan format QR lama dan baru

Statistik komprehensif
Export to Excel/CSV
Persistent data storage

Teknologi: HTML5, CSS3, JavaScript ES6+, jsQR Library (QR Decoding), Canvas API

Kompatibilitas: Desktop & Mobile browsers dengan camera access

Ukuran File: ~35KB (compressed)

Data Storage: LocalStorage dengan auto-backup sistem

üîí Keamanan Data

QR Code Validation: Sistem validasi multi-layer memastikan data QR tidak dapat dimanipulasi. Format QR yang terenkripsi mencegah siswa mengubah nilai ujian.

Duplicate Detection: Otomatis mendeteksi dan mencegah double-scanning untuk siswa yang sama.

Data Integrity Check: Verifikasi checksum pada setiap hasil scan untuk memastikan data tidak rusak.

üìà Fitur Analisis

  • Perhitungan otomatis: Total siswa, rata-rata nilai, nilai tertinggi/terendah
  • Filter dan pencarian data berdasarkan nama, NIS, kelas, atau tanggal
  • Visualisasi distribusi nilai per kelas
  • Export data dalam format Excel untuk analisis lebih lanjut
  • Tracking pelanggaran ujian per siswa
